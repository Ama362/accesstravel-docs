Welcome to the **AccessTravel** documentation portal.

AccessTravel is a SaaS platform designed to help travelers with disabilities and accessibility needs plan, book, and experience trips with confidence across air, sea, and ground transportation.

== What AccessTravel Does

AccessTravel helps users:

* Discover accurate accessibility information before booking
* Compare accessible travel options across transport types
* Reduce uncertainty and travel stress for people with disabilities
* Enable service providers to share accessibility data responsibly

== Who This Documentation Is For

This documentation is intended for:

* Travelers using the AccessTravel platform
* Accessibility advocates and contributors
* Partner transport providers
* Product and support teams
